# Vision review source package

This package contains the **markdown source materials** for the presentation deck, along with the **Python code used to generate the diagrams**.

## Folder structure

- `presentation/`
  - `vision_foundation_models_review.pptx` — the PowerPoint deck.
- `report/`
  - `presentation_report.md` — cleaned markdown report version of the presentation.
  - `slide_text_dump.md` — literal text extraction from the PowerPoint deck.
- `references/`
  - `references.md` — stable bibliographic references only.
  - `references.bib` — BibTeX for the three reviewed papers.
  - `source_evidence.md` — page-level notes showing where the main claims come from.
- `data/`
  - CSV files for the comparison table, capability snapshot, MoGe-2 ablation cues, and decision matrix.
- `code/`
  - Python scripts for regenerating the diagrams.
- `assets/`
  - PNG diagrams generated by the scripts.

## Regenerating the diagrams

From the package root:

```bash
python code/generate_all_diagrams.py --outdir assets
```

## Notes

- The cleaned report keeps only **stable** references in the bibliography.
- The literal slide dump preserves the original slide wording, including any time-sensitive notes that were present in the deck.
- The original paper PDFs are not duplicated here in order to keep the package smaller; this source package assumes you already have the uploaded PDFs separately.
